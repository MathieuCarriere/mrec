Example: mipex3.m

This example illustrates using the Cplex Class API to enter and optimize a mixed integer programming problem with special ordered sets (SOS), demonstrating different methods for creating a problem.

The user has to choose the method via an argument: mipex3('r') generates the problem by adding rows, mipex3('c') generates the problem by adding columns, and mipex3('n') generates the problem by adding a list of coefficients.
