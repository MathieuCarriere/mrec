Example: rates.m

This example illustrates how to solve a planning problem with semi-continuous variables.

Assume you run a power supply company. You have several power generators available, each of which has a minimum and maximum production level and a cost per unit output. The question is which generators to use in order to minimize the overall operation cost while satisfying the demand.

This example reads the data from the file '../data/rates.dat'.
