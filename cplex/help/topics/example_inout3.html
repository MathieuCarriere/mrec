Example: inout3.m

This example illustrates using the Cplex Class API to model and solve a production model.

A company has to produce 3 products, using 2 resources. Each resource has a limited capacity. Each product consumes a given number of machines. Each product has a production cost (the inside cost). Both products can also be brought outside the company at a given cost (the outside cost). The objective is to minimize external production given product demand, a cost constraint, and minimum internal production constraints.
