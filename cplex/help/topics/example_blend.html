Example: blend.m

This example illustrates using the Cplex Class API to model and solve a blending problem.

The goal is to produce an alloy by blending four sources: pure metal, raw materials, scrap, and ingots. Each source has a cost. Each source is made up of elements in different proportions. The alloy has minimum and maximum proportions of each element. The objective is to minimize the cost of producing a requested quantity of alloy.

Ingots are discrete. To model ingots with integer variables and solve as a MIP, use 'i' as an argument. Otherwise, the problem will be solved as an LP and the amount of ingots may be fractional.
