Example: lpex7.m

This example illustrates using the Cplex Class API to read in and optimize a problem as well as print names with the answer.

This is a modification of lpex2.m. To run this example, function arguments are required, i.e., lpex7(filename), where filename is the name of the file, with .mps, .lp, or .sav extension.
