Example: cplexmiqpex.m

This example uses the function cplexmiqp to solve a mixed-integer quadratic programming model.

The MIQP problem solved in this example is:

    Maximize  x1 + 2 x2 + 3 x3 + x4
              - 0.5 ( 33x1*x1 + 22*x2*x2 + 11*x3*x3 - 12*x1*x2 - 23*x2*x3 )
    Subject to
       - x1 +   x2 + x3 + 10 x4 <= 20
         x1 - 3 x2 + x3         <= 30
                x2      - 3.5x4  = 0
    Bounds
         0 <= x1 <= 40
         0 <= x2
         0 <= x3
         2 <= x4 <= 3
    Integers
        x4
